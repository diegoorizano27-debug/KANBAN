# Lean–Digital Kanban Thesis Presentation

Proyecto Vite + React + Tailwind CSS listo para abrir en Visual Studio Code.

## Requisitos
- Node.js 18.2+ recomendado
- npm 9+

## Instalación
```bash
npm install
npm run dev
```

## Producción
```bash
npm run build
npm run preview
```

## Stack
- Vite
- React
- Tailwind CSS
- Motion for React
- Recharts
- Lucide React

## Notas
- El proyecto está preparado como una presentación interactiva de tesis.
- El paquete `motion` es la evolución actual de Framer Motion para React.
- El diseño utiliza fondo claro, navegación lateral, animaciones suaves y visualización interactiva.
